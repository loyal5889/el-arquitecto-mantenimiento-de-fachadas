# Calculadora de Cantidades - Mantenimiento de Fachadas

Este programa en **Python** permite calcular de forma automática las **áreas de muros, fachadas y ventanas** en proyectos de mantenimiento en obras civiles.  
Además, descuenta las áreas de las ventanas de las fachadas correspondientes y **exporta los resultados a Excel**, facilitando la elaboración de presupuestos y control de cantidades de obra.

---

## Funciones principales:
- Cálculo de áreas de muros y fachadas.  
- Registro de ventanas con sus dimensiones y perímetros.  
- Descuento automático del área de ventanas sobre las fachadas.  
- Resumen de resultados por tipo de superficie.  
- Exportación de resultados a **Excel (.xlsx)**.  

---

## Aplicaciones:
Ideal para ***consultorias*** o trabajos en campo que necesiten estimar o verificar cantidades de obra. Facilita el proceso repetitivo en excel y aplica para la mayoria de obras civiles que se ejecutan en propiedad horizontal.  

---

## Tecnologías utilizadas:
- **Python**
- Librerías:
  - `pandas` → para manejo de datos y exportación a Excel
  - `openpyxl` → soporte para escribir archivos .xlsx
  - `os`, `math` → utilidades del sistema y cálculos básicos
